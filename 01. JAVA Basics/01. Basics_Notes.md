## ☕ Complete Master Notes on Java (Fully Combined + In-Depth + Teaching Format)

# 🔹 1. Introduction to Java

## ✅ Definition:

Java is a **high-level, object-oriented, class-based programming language** used to develop applications that are **platform-independent, secure, and efficient**.

---

## 🔬 Deep Explanation

### 🔸 High-Level Language

Java uses human-readable syntax, making it easier than machine-level languages.

```java
System.out.println("Hello World");
```

---

### 🔸 Object-Oriented

Java is based on real-world concepts:

* Class → blueprint
* Object → real entity

👉 Example:
A **Student**:

* name
* age
* marks

---

### 🔸 Platform Independence (WORA)

## Definition:

Write Once, Run Anywhere → same code runs on any system.

---

## 🔬 Working:

1. Write → `.java`
2. Compile → `.class` (bytecode)
3. Run → JVM executes

👉 JVM makes Java platform-independent

---

## 🧠 Analogy:

Java = English
JVM = Translator
OS = Countries

---

# 🔹 2. Structure of Java Program

## ✅ Definition:

A Java program consists of classes and methods, starting execution from `main()`.

---

## Example:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

---

## 🔬 Deep Explanation

---

### 🔸 Class

## Definition:

A class is a **blueprint that holds variables and methods**.

```java
class Student {
    String name;
    int age;
}
```

---

### 🔸 main() Method

## Definition:

Entry point of the program.

---

### Breakdown:

```java
public static void main(String[] args)
```

* public → accessible
* static → no object required
* void → no return
* main → starting point
* String[] args → inputs

---

### 🔸 Output

```java
System.out.println("Hello");
System.out.print("World");
```

---

# 🔹 5. Data Types

A data type in Java defines **what kind of value a variable can store**, how much **memory it occupies**, and what **operations can be performed on it**.

---

When you create a variable, Java must know:

* What type of data it will store (number, text, true/false)
* How much memory to allocate
* What operations are allowed (e.g., math operations only on numbers)

---

## 🔸 Primitive Types

Primitive types are **basic built-in data types** that store actual values.

---

## 🔹 Integer Types

Used to store **whole numbers (without decimal values)**.

---

### Example:

```java
int x = 10;
long y = 100000L;
```

---

* `int` is most commonly used for normal numbers
* `long` is used when numbers are very large

---

### 🧠 Real-Life Example:

```java
int age = 25;
long population = 1400000000L;
```

👉 Age is small → `int`
👉 Population is large → `long`

---

---

## 🔹 Floating Types

### ✅ Definition:

Used to store **decimal numbers (fractional values)**.

---

### Example:

```java
float f = 3.14f;
double d = 3.1415;
```

---

* `float` → less precision (approximate values)
* `double` → more precision (accurate values)

---

### 🧠 Real-Life Example:

```java
double price = 99.99;
float temperature = 36.6f;
```

---

---

## 🔹 Character Type

### ✅ Definition:

Used to store a **single character** enclosed in single quotes.

---

### Example:

```java
char c = 'A';
```

---

### 🔬 Deep Explanation:

* Internally stored as Unicode value
* Each character has a numeric representation

---

### 🧠 Example:

```java
char grade = 'B';
System.out.println(grade);
```

---

---

## 🔹 Boolean Type

### ✅ Definition:

Used to store only two values: **true or false**.

---

### Example:

```java
boolean isTrue = true;
```

---

### 🔬 Deep Explanation:

* Used in decision making
* Forms the base of conditions and logic

---

### 🧠 Example:

```java
boolean isPass = true;

if(isPass){
    System.out.println("Student Passed");
}
```

---

---

## 🔸 Non-Primitive Types

### ✅ Definition:

Non-primitive types store **reference (memory address)** instead of actual values.

---

### Examples:

* String
* Array
* Class

---

# 🔹 6. Operators

## ✅ Definition:

Operators are symbols used to **perform operations on variables and values**.

---

## 🔬 Deep Understanding:

Operators allow Java programs to:

* Perform calculations
* Make decisions
* Compare values
* Update variables

---

## 🔸 Arithmetic Operators

### Definition:

Used for **mathematical calculations**.

---

### Example:

```java
int a = 10, b = 3;

System.out.println(a + b); // 13
System.out.println(a - b); // 7
System.out.println(a * b); // 30
System.out.println(a / b); // 3
System.out.println(a % b); // 1
```

---

### 🔬 Important Concept:

👉 Integer division removes decimal part

---

---

## 🔸 Unary Operators

### Definition:

Operate on a **single variable**.

---

### Example:

```java
int x = 5;

System.out.println(x++); // 5
System.out.println(++x); // 7
```

---

### 🔬 Deep Explanation:

| Type | Meaning                  |
| ---- | ------------------------ |
| x++  | use first, then increase |
| ++x  | increase first, then use |

---

---

## 🔸 Relational Operators

### Definition:

Used to **compare two values** and return true/false.

---

### Example:

```java
System.out.println(10 > 5);  // true
System.out.println(10 == 5); // false
```

---

---

## 🔸 Logical Operators

### Definition:

Used to combine multiple conditions.

---

### Example:

```java
boolean a = true, b = false;

System.out.println(a && b); // false
System.out.println(a || b); // true
```

---

### 🔬 Deep Concept:

* AND (`&&`) → both must be true
* OR (`||`) → at least one true

---

---

## 🔸 Assignment Operators

### Definition:

Used to assign values to variables.

---

### Example:

```java
int x = 10;

x += 5; // 15
x -= 2; // 13
```

---

# 🔹 7. Strings

## ✅ Definition:

A String is an object used to **store a sequence of characters (text)**.

---

## 🔬 Deep Explanation:

* String is NOT primitive
* It is a class
* Stored in **String Pool**
* Immutable

---

## 🔸 Immutable Concept

### Definition:

Once a string is created, its value **cannot be changed**.

---

### Example:

```java
String s = "Hello";
s = s + " World";
```

👉 This creates a **new object**, not modify old one

---

---

## 🔸 Example:

```java
String name = "Java";
System.out.println(name);
```

---

---

## 🔸 Methods (Detailed)

```java
String s = "Hello";
```

---

### length()

```java
System.out.println(s.length()); // 5
```

---

### charAt()

```java
System.out.println(s.charAt(1)); // e
```

---

### substring()

```java
System.out.println(s.substring(0,2)); // He
```

---

### equals()

```java
System.out.println(s.equals("Hello")); // true
```

---

---

## ⚠️ Important Concept

```java
String a = new String("Hello");
String b = new String("Hello");

System.out.println(a == b); // false
System.out.println(a.equals(b)); // true
```

---

## 🔬 Explanation:

* `==` → compares memory
* `.equals()` → compares value

---

# 🔹 8. Type Casting

## ✅ Definition:

Type casting is the process of **converting one data type into another**.

---

## 🔸 Implicit Casting

```java
int a = 10;
double b = a;
```

👉 automatic conversion

---

## 🔸 Explicit Casting

```java
double d = 10.5;
int x = (int)d;
```

👉 manual conversion (data loss possible)

---

# 🔹 9. Control Statements

## ✅ Definition:

Control statements are used to **control the flow of execution**.

---

## 🔸 if-else

```java
int marks = 85;

if(marks >= 50){
    System.out.println("Pass");
} else {
    System.out.println("Fail");
}
```

---

### 🔬 Deep Explanation:

* Condition is checked
* Based on result → block executes

---

---

## 🔸 switch

```java
int day = 1;

switch(day){
    case 1: System.out.println("Monday"); break;
    case 2: System.out.println("Tuesday"); break;
}
```

---

# 🔹 10. Loops

## ✅ Definition:

Loops are used to **repeat a block of code multiple times**.

---

## 🔸 for Loop

```java
for(int i=0; i<5; i++){
    System.out.println(i);
}
```

---

### 🔬 Explanation:

* Initialization → start
* Condition → stop
* Increment → update

---

---

## 🔸 while Loop

```java
int i = 0;

while(i < 5){
    System.out.println(i);
    i++;
}
```

---

# 🔹 11. Arrays

## ✅ Definition:

An array is a collection of elements of the **same data type stored in contiguous memory locations**.

---

## Example:

```java
int[] arr = {1,2,3};

System.out.println(arr[0]); // 1
```

---

## 🔬 Explanation:

* Index starts from 0
* Fixed size

---

---

## Example:

static void greet(){
